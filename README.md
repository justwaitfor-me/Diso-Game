# DiscoGame

DiscoGame is a Python terminal roleplay game!

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install DiscoGame.

```bash
pip install DiscoGame
```

## Usage

```python
import discogame

discogame.play()
```
